Connect Remote Files to GitHub
Create a localhost server using node.js

Step 1

Run the command "git init" (in the folder you want to upload on git hub).

This makes your current folder the master

Step 2

Run the command "ls" to check that the files are correct.

Step 3

Run the command "git commit-m 'Type message here'"

Step 4

Run the command "git add fileName"

Step 5

Create a new Repo on GitHub. You'll get a link like this
"git remote add origin https://github.com/vikrantRajan/NewProject2.git"
This connects your computer to github.

Step 6

Run the command "git push -u origin master"
This will push files onto GitHub.

Step 7

type this in JavaScript
"var express = require("express");
var app = express();
app.use(express.static(__dirname));
app.listen(2000, function()
{ console.log(`server started on port 2000`); });"

Step 8

Open the terminal in vsCode (or open another gitBash in the same file location for node.js). Run the command "npm install"

Step 9

Run the command "node server". This will start a new node server with localhost:2000 (because of the JavaScript Code written above.)
